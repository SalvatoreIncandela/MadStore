Support

GridGain comes with free community support and available commercial support that is geared towards the use and development in commercial environment:

Community Support

You have three resources to obtain free community support:

  • Use public issue tracking system

Additionally to a free community support, GridGain Systems provides number of services targeting the usage of GridGain product in commercial environment:
  • Support

Updates

You can periodically check www.gridgain.com for updates or simply watch for log message when running GridGain.
GridGain automatically checks for new version each time it starts and notifies you if updates are available.

Uninstall

GridGain installers do not modify the system except for the installed files.
You can always just remove installation folder to remove GridGain.
You can also use graphical uninstaller if you used graphical installer. Refer to installation instructions for more details.
